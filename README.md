# VPS 自动化脚本集合

实用的 VPS 管理脚本集合，帮助你快速部署和管理服务器。

## 脚本列表

### 系统管理
- **docker-manager.sh** - Docker 容器管理工具 (start/stop/restart/status/logs/clean)
- **mysql-backup.sh** - MySQL 数据库自动备份脚本 (支持云端同步)

### VPN 工具
- 一键 WireGuard 安装
- 一键 OpenVPN 安装
- IPSec VPN 配置

### 安全加固
- SSH 安全配置
- 防火墙设置
- 自动更新脚本

## 快速使用

```bash
git clone https://github.com/clashforwindows-net/vps-scripts.git
bash scripts/docker-manager.sh status
```
